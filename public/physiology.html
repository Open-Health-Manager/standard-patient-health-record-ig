Recurring challenges with designing PHR apps include figuring out which data to include (all of it? just the hospital data? fitness data?) and then finding suitable test data. This implementation guide does not purport to provide algorithms or workflows for every medical category. However, we can reference the Synthea synthetic data generator, and recommend that PHR developers use itâ€™s algorithms for generating sample data for various medical conditions.

# go into the cloned directory
cd synthea

# build the app
./gradlew build check test
